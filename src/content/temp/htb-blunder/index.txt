## Overview

Blunder is an easy-rated Linux box on HackTheBox. The attack chain involves:

1. Directory enumeration → find a Bludit CMS installation
2. CSRF bypass on the login brute-force protection
3. Image upload RCE via CVE-2019-16113
4. Lateral movement from `www-data` → `hugo`
5. Privilege escalation via `sudo` bypass (CVE-2019-14287)

<Callout type="tip" title="Box Info">
- **OS**: Linux
- **Difficulty**: Easy
- **IP**: 10.10.10.191
</Callout>
